# AISRT 文档 Wiki：本地 AI 字幕生成、SRT 翻译与发布维护

本地 AI 字幕生成、音视频转 SRT、多语言语音识别、字幕时间轴对齐、SRT 字幕翻译、离线字幕工具、GUI 字幕软件、CLI 字幕批处理、Windows portable 发布、开源发布检查。
