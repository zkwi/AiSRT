# 支持说明

AISRT 是个人维护的本地桌面/命令行工具，不提供商业支持或私下调试服务。公开问题请优先使用 GitHub Issues。

## 提问前

- 先运行 `ai-sub doctor`，确认 FFmpeg、CUDA/PyTorch、输出目录和模型缓存状态。
- 搜索已有 Issue，避免重复报告。
- 准备操作系统、Python 版本、是否使用 GPU、运行命令或界面操作步骤。
- 删除日志中的个人路径、真实媒体名、邮箱、Token、Cookie、API Key 和完整模型缓存路径。

## 可以提交 Issue 的情况

- 稳定复现的崩溃、报错或字幕输出异常。
- README 或安装步骤与实际行为不一致。
- GUI/CLI 中影响普通使用的体验问题。
- 不增加 Web UI、服务端、数据库、账号系统或在线任务队列的实用改进建议。

## 不适合提交的内容

- 未授权公开的媒体文件、字幕、截图或完整日志。
- 要求接入第三方翻译 API、本地外的托管服务或账号系统。
- 与本项目无关的模型训练、硬件采购或通用 Python 环境咨询。

安全问题请按 [SECURITY.md](SECURITY.md) 说明处理。
